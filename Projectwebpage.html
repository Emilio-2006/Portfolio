Article Tag The article tag specifies independent, self-contained content.

Section Tag The section element is a structural HTML element used to group together related elements.

Aside Tag The aside element represents a section of a page that consists of content that is tangentially related to the content around the aside element, and which could be considered separate from that content.

Figure Tag The figure tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

Figure Caption Tag The figcaption element is used as a child of a parent figure element to attach a caption to the image, table, or chart contained in the figure element.

Video Embed Tag The video tag is used to embed video content in a document, such as a movie clip or other video streams.

Audio Embed Tag The audio tag is an inline element that is used to embed sound files into a web page.

Video and Audio Source Tag The video tag is used to embed video content in a document, such as a movie clip or other video streams. The audio tag is used to embed sound content in a document, such as music or other audio streams.

dl Tag The dl HTML element represents a description list. The element encloses a list of groups of terms (specified using the dt element) and descriptions (provided by dd elements).

dt Tag The dt HTML element specifies a term in a description or definition list, and as such must be used inside a dl element.

tag. This tag, standing for ' definition description ', is used to provide a description or definition of an item in a description list.

optgroup Tag The optgroup HTML element creates a grouping of options within a select element.
