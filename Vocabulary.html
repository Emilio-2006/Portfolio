Vocabulary page

Layer A layer communicates parts if the software.
Software a set of instructions, data or programs used to operate computers and execute specific tasks.
Logic Layer the part of a program where classes determine the information needed to solve problems
Authentication Verifying the identity of a user, process, or device, often as a prerequisite to allowing access to resources in an information system.
Persistence Layer A software system that is similar to a database and saves/gets data from a storage
Error-Messaging An error message is the information displayed when an unforeseen problem occurs, usually on a computer or other device.
Authorization the process of giving someone the ability to access a resource
User Interface The user interface (UI) is the point of human-computer interaction and communication in a device.
Application Software Is a program designed to handle specific tasks for users.
Cross-cutting concerns is a concern where it affects multiple layer of code.
Error handling The process of responding to and recovering from error conditions in your program.
logging Logging is the act of keeping a log of events that occur in a computer system, such as problems, errors or just information on current operations.
